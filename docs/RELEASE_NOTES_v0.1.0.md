# HNS Investments v0.1.0

First public prerelease of HNS Investments, a read-only desktop companion for Handshake portfolio review.

## Highlights

- Connects to bridge-enabled Bob LearnHNS for automatic Domains, Coins, and Shakedex data.
- Uses a local app support manifest with connection details from Bob LearnHNS.
- Keeps wallet interaction read-only.
- Does not sign transactions, import seeds, mutate wallets, or export private keys.
- Hides USD values by default.
- Marketplace and Funding views are planned for future releases.

## macOS Release Verification

This prerelease includes signed, notarized, stapled, and Gatekeeper-verified macOS DMGs for Apple Silicon and Intel Macs. Windows and Linux builds are also included for internal testing.

Included assets:

- `HNS-Investments-0.1.0-arm64.dmg`
- `HNS-Investments-0.1.0-x64.dmg`
- `HNS-Investments-0.1.0-x64.exe`
- `HNS-Investments-0.1.0-x86_64.AppImage`
- `SHA256SUMS.txt`

## Notes

This is an early prerelease for testing with Bob LearnHNS bridge support. Use it with non-sensitive review workflows first and report any connection or display issues before relying on it for regular portfolio review.
